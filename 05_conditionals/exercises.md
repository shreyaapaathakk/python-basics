# Conditional Statements - Exercises

Complete these exercises before checking `practice.py`.

---

## Exercise 1

Write a program to check whether a person is eligible to vote.

Condition:

- Age ≥ 18

---

## Exercise 2

Write a program to determine whether a number is even or odd.

---

## Exercise 3

Write a program to find the larger of two numbers.

---

## Exercise 4

Write a program to assign grades based on marks.

| Marks | Grade |
|--------|-------|
| 90–100 | A |
| 75–89 | B |
| 60–74 | C |
| 40–59 | D |
| Below 40 | F |

---

## Exercise 5

Create a login system.

If the username is `"admin"` and the password is `"python123"`,

display:

```text
Login Successful
```

Otherwise display:

```text
Invalid Credentials
```

## Exercise 6

## Exercise 7
