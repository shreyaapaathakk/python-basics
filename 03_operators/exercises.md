# Operators - Exercises

Complete these exercises before checking `practice.py`.

---

## Exercise 1

Create two variables and perform:

- Addition
- Subtraction
- Multiplication
- Division
- Floor Division
- Modulus
- Exponentiation

---

## Exercise 2

Create a variable named `count` with the value `10`.

Update it using:

- `+=`
- `-=`
- `*=`
- `/=`

Print the value after each operation.

---

## Exercise 3

Compare two numbers using:

- `==`
- `!=`
- `>`
- `<`
- `>=`
- `<=`

What does each comparison return?

---

## Exercise 4

Create two Boolean variables.

Use:

- `and`
- `or`
- `not`

Print the results.

---

## Exercise 5

Create two lists.

One list should reference the other.

Create a third list with the same values.

Use both `is` and `==` to compare them.

Can you explain the difference?

---

## Exercise 6

Create a list of five fruits.

Check whether:

- `"Apple"` exists in the list.
- `"Orange"` exists in the list.

Use `in` and `not in`.

---

## Exercise 7

Create a program that checks whether a student is eligible for an exam.

Conditions:

- Attendance ≥ 75%
- Fees paid

Use logical operators to determine eligibility.

---

## Bonus Challenge

Ask the user to enter two numbers.

Display:

- Arithmetic operations
- Comparison results
- Whether both numbers are positive
