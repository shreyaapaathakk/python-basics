# Data Types - Exercises

Solve these exercises on your own before checking `practice.py`.

---

## Exercise 1

Create variables to store:

- Your name
- Your age
- Your height
- Whether you are a student

Print the value and its data type.

---

## Exercise 2

Create variables with the following values:

- `25`
- `15.5`
- `"Python"`
- `False`

Use the `type()` function to identify each data type.

---

## Exercise 3

Convert the following:

- `"100"` → Integer
- `50` → Float
- `99.99` → Integer
- `123` → String

Print the converted values and their data types.

---

## Exercise 4

Predict the output before running the program.

```python
number = "25"

print(type(number))
print(type(int(number)))
```

Were your predictions correct?

---

## Exercise 5

Explore the `bool()` function.

Find the result of:

```python
bool(0)
bool(100)
bool("")
bool("Python")
bool([])
bool([1, 2])
```

---

## Exercise 6

Take the user's age as input.

Convert it into an integer and print:

- The value
- Its data type

## Exercise 7

Create variables of all four basic data types.

Display them in this format:

```text
Age: 20 (<class 'int'>)
Height: 5.6 (<class 'float'>)
Name: Alice (<class 'str'>)
Student: True (<class 'bool'>)
```
