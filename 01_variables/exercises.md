# Variables - Exercises

Practice these exercises on your own before looking at the solutions in `practice.py`.

---

## Exercise 1: Personal Information

Create variables to store the following information:

- Your name
- Your age
- Your city
- Your favorite programming language

Print all the values.

### Expected Output

```text
Name: Shreya
Age: 22
City: Varanasi
Favorite Language: Python
```

## Exercise 2: Student Record

Create variables to store:

- Student name
- Roll number
- Course
- Semester
- Percentage

Print the information in a neat format.

### Example Output

```text
===== Student Record =====
Name       : Rahul
Roll Number: 1023
Course     : BCA
Semester   : 3
Percentage : 89.5
```

## Exercise 3: Swap Two Variables

Create two variables with different values.

## Exercise 4: Store Different Data Types

Create variables to store:

- An integer
- A float
- A string
- A boolean

Print both the value and its data type using the `type()` function.

### Example Output

```text
25 <class 'int'>
5.8 <class 'float'>
Python <class 'str'>
True <class 'bool'>
```

## Exercise 5: Rectangle Details

Create variables for the length and width of a rectangle.

Print:

- Length
- Width
- Area (`length × width`)
- Perimeter (`2 × (length + width)`)

### Example Output

```text
Length: 10
Width: 5
Area: 50
Perimeter: 30
```
